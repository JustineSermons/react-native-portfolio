Menu App

React Menu App displaying a restaurant menu page with different sections containing multiple menu items by utilizing arrays, functions, and React components.

  • Built with: React
  • Features: Displays a list of menu items organized under sections, dynamic rendering of menu items using given data, reusable components for menu sections and menu items
  • Skills: Prop handling, component reusage, map method usage
  • Outcome: This project enhanced my skills in React allowing me to gain deeper knowledge and understanding of the map method, handling props, rendering data using an array, and using components.

Exercise App

Exercise App Design in React Native utilizing arrays, navigation, incrementation, and timers to allow users to do repetition and duration exercises.

  • Built with: React Native, Expo
  • Features: Displays suggested exercises to users, increases rep count of exercises, tracks duration of exercises
  • Skills: React Native component usage, navigation to different screens, state management for updates in real time
  • Outcome: This project deepened my understanding of React Native Elements and Navigation Stack to smoothly transition into different unique screens with different purposes for exercising.

Quiz App

React Native Quiz App that supports different question types such as True/False, Multiple Choice, and Multiple Answer. The app utilizes React Navigation and users can receive a score upon quiz completion in a Summary screen displaying both the correct answers and total score.

  • Built with: React Native, Expo
  • Features: Renders questions and answers to the user, allows users to select answers and progress to next questions, displays a summary screen at the end for users to see results, navigation.
  • Skills: React Navigation to go to different screens, React Native Element usage, state management for updates in real time
  • Outcome: This project helped me gain experience in developing interactive quiz applications, state management, and navigation.

To-Do App

React Native To-do App that allows users to add and complete tasks, login, register, and type information that validates and stores their input using regular expressions and storage.

  • Built with: React Native, Expo
  • Features: Login interface allowing users to input a username and password that navigates them to the Todo screen upon successful authentication, registration screen allowing users to enter information that gets validated and stored, add and complete tasks in a list, regular expression usage for input field validation, and navigation to different screens
  • Skills: Regular expressions for input validation to ensure data integrity and security, React Navigation to navigate to different screens, async storage usage to store user information
  • Outcome: This project enhanced my knowledge of React Native elements, input validation, and storing user data which is very important for a secure user experience.
